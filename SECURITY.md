# Security Notes

SF Release Update is a public static dashboard generated by a scheduled scraper. The scraper reads public SAP What's New Viewer pages and may optionally send notifications through Telegram.

## Secrets

- Store `TELEGRAM_BOT_TOKEN` and `TELEGRAM_CHAT_ID` only as GitHub Actions secrets.
- Do not print secrets, private channel IDs, or internal testing notes in workflow logs.
- Rotate Telegram tokens immediately if they are exposed.

## Data Freshness

- Treat the generated `data/updates.json` as a snapshot, not an official SAP record.
- Review scraper failures quickly so the public dashboard does not silently become stale.
- Keep direct SAP Help links in the UI so users can verify any item against the source.
